# Conditional Operators

## AND: &&
```java
true && true
// true
false && true
// false
true && false
// false
false && false
// false
```

## OR: ||
```java
true || true
// true
false || true
// true
true || false
// true
false || false
// false
```

## NOT: !
```java
!false
// true
!true
// false
```
